Matching Socks CUDA path tracer 
by Samuel Lapere, 2016
https://raytracey.blogspot.com

Demo of high performance CUDA accelerated path tracing based on the GPU ray tracing framework of 
Timo Aila, Samuli Laine and Tero Karras (Nvidia Research)
https://research.nvidia.com/publication/understanding-efficiency-ray-traversal-gpus-kepler-and-fermi-addendum
https://mediatech.aalto.fi/~timo/HPG2009/

Source code available at https://github.com/straaljager/GPU-path-tracing-tutorial-4

HDR map from http://www.hdrlabs.com/sibl/archive.html

User interaction:

- Left mouse button: rotate scene	
- Right mouse button: zoom

- G/H: change aperture
- T/Y: change focal distance
- W, A, S, D, F, R, arrow keys: move camera
